# 1.项目介绍
## 1.1 概述
**声网实验室**
声网是实时互动行业的开创者和引领者，在全球范围内最早定义了实时互动云服务的业务方式。声网实验室为声网展示各项RTC技术能力的应用，用户可以深度了解并体验声网各项音视频技术能力。
